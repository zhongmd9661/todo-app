# 待办事项应用 (Todo App)

一个简单而实用的待办事项管理应用，使用原生 HTML、CSS 和 JavaScript 构建。

## 功能特性

- ✅ 添加待办事项（支持点击按钮和回车键）
- ✅ 标记任务完成/未完成
- ✅ 删除任务
- ✅ 实时显示未完成任务计数
- ✅ 数据持久化（使用 localStorage）
- ✅ 响应式设计

## 技术栈

- 原生 HTML5
- 原生 CSS3
- 原生 JavaScript (ES6+)
- localStorage API

## 使用方法

1. 克隆或下载此仓库
2. 在浏览器中打开 `index.html` 文件
3. 开始使用待办事项应用

## 文件结构

```
todo-app/
├── index.html      # 主页面
├── script.js       # 应用逻辑
├── style.css       # 样式文件
├── test-plan.md    # 测试计划
└── README.md       # 项目说明
```

## 浏览器支持

- Chrome (推荐)
- Firefox
- Safari
- Edge

## 开发

### 本地运行

```bash
# 使用 Python 启动简单服务器
python -m http.server 8000

# 或使用 Node.js
npx serve
```

然后在浏览器中访问：http://localhost:8000
